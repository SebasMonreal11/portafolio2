Objetivos

Objetivo general

Desarrollar una página web con una base de datos integrada, con el propósito principal de aprender y dominar las herramientas y tecnologías necesarias para su implementación.

Objetivos especificos

Desarrollar una página web con una base de datos integrada, con el propósito principal de aprender y dominar las herramientas y tecnologías necesarias para su implementación.

Diseño de la pagina web

Aprender y aplicar los principios de diseño web para crear una interfaz de usuario atractiva y funcional.

Desarrollo de la base de datos

Adquirir conocimientos sobre el diseño y la gestión de bases de datos. Esto incluirá la creación de tablas, la definición de relaciones entre ellas y la implementación de consultas para manipular los datos.

Integracion de la base de datos con la página web

Aprender a conectar la base de datos con la página web para que los datos puedan ser accedidos y manipulados a través de la interfaz de usuario.

Seguridad de la pagina web y la base de datos

Comprender y aplicar las mejores prácticas de seguridad para proteger la página web y la base de datos contra posibles amenazas.

Pruebas y depuracion

Aprender a realizar pruebas y depurar el código para garantizar que la página web y la base de datos funcionen correctamente.
